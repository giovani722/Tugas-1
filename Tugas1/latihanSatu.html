Operating System Windows

Operating System Windows buatan Microsoft yang dirilis pada 1985, dengan antarmuka grafis yang mudah digunakan. Umumnya digunakan di komputer pribadi (PC) dan banyak perangkat lainnya.

Operating System Linux

Sistem operasi open-source berbasis kernel Linux, dirilis pada 1991. Dapat diubah dan didistribusikan bebas, populer untuk server, tapi juga digunakan di desktop dan perangkat lain.
